📋 Описание проекта

Учебный проект интернет-магазина, разработанный в рамках курса "Фронтенд и бэкенд разработка". 
Представляет собой полноценное fullstack-приложение с возможностью просмотра, добавления, 
редактирования и удаления товаров.

✨ Функциональность
- 📦 Просмотр списка товаров
- ➕ Добавление новых товаров
- ✏️ Редактирование существующих
- 🗑️ Удаление товаров
- 🖼️ Отображение изображений товаров
- 📱 Адаптивный дизайн

🎯 Выполненные практические работы

| № | Тема | Реализация |
|---|------|------------|
| **1** | CSS-препроцессоры | Переменные SASS, миксины, вложенные селекторы, светлая тема |
| **2** | Сервер на Express | REST API, CRUD операции, middleware логирования |
| **3** | JSON и API | JSON формат, HTTP статусы, валидация данных |
| **4** | API + React | Интеграция с axios, CORS, асинхронные запросы |
| **5** | Расширенный REST API | RESTful маршруты, правильные методы (GET, POST, PATCH, DELETE) |
| **6** | Подготовка к КР | README, структура проекта, документация |
